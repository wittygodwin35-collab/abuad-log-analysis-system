# Upload Test Log Samples

`loghub-linux-2k.log` is an actual Linux system log sample from the public Loghub dataset:

- Source: https://github.com/logpai/loghub/blob/master/Linux/Linux_2k.log
- Dataset homepage: https://github.com/logpai/loghub
- Access terms: Loghub describes the datasets as freely available for research or academic work and asks users to cite the Loghub paper where applicable.

The smaller `abuad-upload-sample.log` file is a synthetic app smoke-test fixture. Use `loghub-linux-2k.log` when you need a real open-access upload file.
